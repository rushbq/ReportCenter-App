# ReportCenter App

企業報表中心，提供各部門營運數據視覺化儀表板，包含採購、業務、財務、人資、資訊等部門報表管理與查閱。

## 功能

- 營運總覽儀表板（KPI、營收趨勢、部門比較）
- 部門報表瀏覽（卡片 / 表格檢視切換）
- 報表明細頁（圖表、篩選、資料表格）
- 快速存取與收藏管理

## 技術

- React 18
- Vite
- Recharts（圖表）
- Lucide React（圖示）

## 開發

```bash
npm install
npm run dev
```

## 建置

```bash
npm run build
```
